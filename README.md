# 🎨 BDA Audit Bancaire - Frontend

## 📋 Description
Interface utilisateur React pour la gestion bancaire.

## 🚀 Technologies
- **React 18** - UI
- **Vite** - Build tool
- **Material UI** - Composants
- **Axios** - API
- **Recharts** - Graphiques

## ⚙️ Installation

```bash
# 1. Cloner le projet
git clone https://github.com/RAZAFITSALAMA-sandra/frontend_bda.git
cd frontend_bda

# 2. Installer les dépendances
npm install

# 3. Configurer l'environnement
cp .env.example .env

# 4. Démarrer l'application
npm run dev
